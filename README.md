# Taiwan Biotech Stock & TFDA License Automation Matcher (臺灣生技醫藥上市/上櫃/興櫃成分股 — TFDA 藥證許可證自動化比對系統)

本專案是一個跨領域的資料自動化處理工具，旨在協助生技醫藥產業研究人員、證券分析師或法規事務（RA）從業者。系統能自動化抓取臺灣股票市場最新生技醫療成分股，並與臺灣食藥署（TFDA）官方的西藥許可證開放資料庫進行本地端的大數據模糊比對，最終導出易於分析的 Excel 雙工作表（Sheets）報表。

## 🚀 核心功能與流程拆解

1. **生技成分股動態擷取 (Step 1)**
   - 透過 Python `Selenium` 與 `BeautifulSoup4` 爬蟲技術，動態擷取公開市場（上市、上櫃、興櫃）最新的生技醫療產業成分股。
   - 內建**字串清洗算法**：自動剔除股票市場特有的名稱後綴（如 `-KY`、`KY`、`創`、`*`、連字號與市場代碼），還原出精準的生技公司中文核心名稱。

2. **TFDA 開放資料極速匹配 (Step 2)**
   - 整合內建 `Pandas` 資料分析引擎，讀取食藥署官方藥證許可證巨量資料庫（幾十萬筆等級）。
   - 利用正則表達式（Regex OR）實作本地端的**模糊匹配（Fuzzy Matching）**，在一秒內精準過濾出所有屬於這 60 多家生技公司的現存藥證細節。

3. **雙工作表自動化匯出**
   - **Sheet 1 (`藥證詳細資料`)**: 完整導出包含許可證字號、適應症、劑型、申請商名稱、製造廠等完整 TFDA 官方原始欄位。
   - **Sheet 2 (`有藥證公司代碼`)**: 獨立萃取出成功匹配到藥證的公司，並對照其**股票代碼**，方便使用者後續進行 `VLOOKUP`、`XLOOKUP` 或與其他財務/產業檔案連動比較。
