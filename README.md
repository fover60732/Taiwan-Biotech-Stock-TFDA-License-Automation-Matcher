# Taiwan Biotech Stock & TFDA License Automation Matcher (臺灣生技醫藥上市/上櫃/興櫃成分股 — TFDA 藥證許可證自動化比對系統)

本專案是一個跨領域的資料自動化處理工具，旨在協助生技醫藥產業研究人員、證券分析師或法規事務（RA）從業者。系統能自動化抓取臺灣股票市場最新生技醫療成分股，並與臺灣食藥署（TFDA）官方的西藥許可證開放資料庫進行本地端的大數據模糊比對，最終導出易於分析的工作表報表。

## 🚀 核心功能與流程拆解

1. **生技成分股動態擷取 (Step 1)**
   - 透過 Python `Selenium` 與 `BeautifulSoup4` 爬蟲技術，動態擷取公開市場（上市、上櫃、興櫃）最新的生技醫療產業成分股。
   - 內建**字串清洗算法**：自動剔除股票市場特有的名稱後綴（如 `-KY`、`KY`、`創`、`*`、連字號與市場代碼），還原出精準的生技公司中文核心名稱。

2. **TFDA 開放資料極速匹配 (Step 2)**
   - 整合內建 `Pandas` 資料分析引擎，讀取食藥署官方藥證許可證巨量資料庫（幾十萬筆等級）。
   - 利用正則表達式（Regex OR）實作本地端的**模糊匹配（Fuzzy Matching）**，在一秒內精準過濾出所有屬於Step 1篩出的生技公司藥證細節。

### 3. **雙版本輸出支援 (二選一)**
- **本地指令版 (`main.py`)**：自動匯出包含雙工作表的 Excel 檔案。
  - **Sheet 1 (`藥證詳細資料`)**：完整導出包含許可證號、適應症、劑型、申請商名稱、製造廠等完整 TFDA 官方原始欄位。
  - **Sheet 2 (`有藥證公司代碼`)**：獨立萃取出成功匹配到藥證的公司，對照其**股票代碼**，方便後續進行 `VLOOKUP`、`XLOOKUP` 或與其他財務/產業檔案連動比較。
- **網頁互動版 (`st.2.py` - 無二進位安全模式)**：
  - 提供美觀的網頁操作介面。為繞過企業/醫院資安封鎖，不載入 numpy/pandas 等二進位組件，改為直接於網頁提供 **兩個獨立的 CSV 檔案** 點擊下載（有藥證之公司代碼清單、藥證詳細資料）。

## 🔒 獨家特色：無二進位安全模式 (Pure Python Mode)
為了應對企業內網或醫院極為嚴格的 IT 資安防護原則（如 Windows Defender 應用程式控制 / AppLocker 封鎖暫存區二進位 DLL 檔案之限制），本專案特別提供**網頁版安全模式**：
- **零外接 DLL 依賴**：完全拔除 `pandas` 與 `numpy` 等高度依賴 C 語言二進位編譯的套件。
- **純 Python 內建驅動**：核心比對邏輯與檔案串流（I/O）皆採用 Python 內建標準庫與純文字處理，確保在極度嚴苛的資安環境下仍可流暢運作。

## 💡 設計目的與應用場景 (Conflict of Interest Declaration)
本工具在設計上特別考量了醫藥產業與學術界的合規需求：
1. **學術與臨床聲明**：提供醫藥產業研究人員、臨床專家及學術發表者，在進行利益衝突聲明（Conflict of Interest）時之關鍵參考，協助確認並宣告自身無特定持股生技公司之藥證關聯性。
2. **合規與自我審查**：協助需要進行合規申報或利益衝突（Conflict of Interest）自我審查的專業人士，快速比對自身未持有任何具備特定 TFDA 藥證的生技公司股份。
